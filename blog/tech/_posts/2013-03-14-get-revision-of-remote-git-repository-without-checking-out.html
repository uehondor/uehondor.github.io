--- layout: post published: true title: Get revision of remote git repository without checking out date: '2013-03-14 10:35:16 +0000' ---

You can get the revision of a remote git repository by using the git ls-remote command

This simple example will output the revision for all branches and tags.

{% highlight bash %}git ls-remote url-to-git-repo:username/repo.git{% endhighlight %}

Output will look something like this:

{% highlight bash %} 8eef34582a... HEAD 4f4d5e18fd... refs/heads/master 84ae6b827a... refs/heads/another-branch ... {% endhighlight %}

You can narrow the output down to just one branch by doing the following

{% highlight bash %}git ls-remote url-to-git-repo:username/repo.git branch-name{% endhighlight %}
